Project: Program and Institutional Outcomes Assessment System

This project delivers an automated system to evaluate academic performance across course, program, and institutional levels using structured input files, outcome mappings, and AI-generated insights. I contributed to every major phase of this system: from backend data processing and validation to frontend GUI and CrewAI-based assessment agents. The outcome is a deployable, scalable tool that can help academic institutions generate actionable reports for accreditation and curriculum analysis.

Direct Contribution Highlights

  • Built the config and input validation module to handle JSON and Excel formats with edge case error handling.
  • Designed the logic to compute Program Outcomes (POs) from Course Outcomes (COs) using mapping specifications.
  • Developed four CrewAI assessment agents to provide insights across CO, PO, IO, and individual student metrics.
  • Engineered the Panel GUI for file uploads, visualization, filtering, and Excel downloads using Plotly and Panel.
  • Implemented extensive testing using Pytest, ensuring system integrity and GUI responsiveness.
  • Completed deployment and authored user and developer documentation for seamless adoption.
